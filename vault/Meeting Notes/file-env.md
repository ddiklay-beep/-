# .env

> [!warning] קובץ סודות
> - **נתיב:** `.env` (שורש הפרויקט)
> - **בעלות:** צוות (פרטי לכל מפתח על המכונה שלו)
> - **סוג:** משתני סביבה
> - **מצב גיט:** **gitignored** — לעולם לא עולה ל-GitHub

## תפקיד
הקובץ הזה מחזיק ערכי סוד אמיתיים למשתני הסביבה של הפרויקט (מפתחות API, וכו'). מכיל את אותם המפתחות שמופיעים ב-[[file-env-example]] רק עם ערכים אמיתיים. כל מפתח על מכונה משלו ממלא אותו בעצמו.

## תוכן עיקרי
מפתחות שיכולים להופיע כאן בעתיד:
- `ANTHROPIC_API_KEY` — מפתח API של Anthropic (אם מישהו מהצוות קורא ל-API ישירות)
- `ANTHROPIC_MODEL` — מודל ברירת-מחדל לסוכנים

## קבצים קשורים
- [[file-env-example]] — תבנית גלויה ל-Git שמכילה את אותו מבנה ללא ערכים
- [[file-gitignore]] — הקובץ שמחריג את `.env` מהמעקב של git
- [[file-claude-md]] — ראובן יכול להחליט מתי הצוות צריך גישה למשתנה כלשהו
